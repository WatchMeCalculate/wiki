Vim
===



## Comment out / Uncomment code
### Comment out
1. `ctrl-v` -> visual block mode
2. select lines
3. `shift + I` -> insert mode
4. add what characters you want
5. press `esc`

### Comment In
1. `ctrl-v` -> visual block mode
2. select lines
3. use arrows to select text, `shift <-/->` selects chunks
5. press `d` or `x` to delete
